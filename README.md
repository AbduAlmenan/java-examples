# java-examples
Примеры на Java. Включают темы ООП, технологии Java, Spring. Все проекты созданы в [NetBeans](netbeans.org).

Хронологический поряток:
* HelloWorldApp
* GeometryApp
* JavaPrg2
* JavaCalc
* JavaGC
* TreeBuilder
* GeometryInheritance
* GeometryInterface
* JavaExceptions
* JavaEnums
* JavaPolymorphism
* JavaCollections
* JavaGenerics
* JavaConfig
* JavaJNA
* JavaSockets
* JavaRMI
* JavaDatabase
* JavaHibernate
* JavaSwing
* JavaWebApp
* JUnitDemo
* SpringDI
* SpringAOP
* SpringJDBC
* SpringScheduler
* SpringSecurity
* SpringMVC
